# Cloud IVR & Contact Center Health Dashboard

# Overview
This project simulates a cloud-based contact center system similar to Genesys Cloud.  
It demonstrates IVR workflow design, call routing logic, queue management and KPI-based performance dashboards.

The project is built to showcase practical understanding of contact center technologies and analytics for roles like Deloitte CS&D – Genesys Analyst.

---

# Features

- Interactive IVR flow simulation  
- Skill-based routing to different queues  
- Sales and Support call segregation  
- Fallback routing for unhandled inputs  
- Real-time KPI dashboard  
- Monitoring of contact center performance

---

# Key Metrics Tracked

- AHT – Average Handle Time  
- SLA – Service Level Agreement  
- FCR – First Call Resolution  
- Abandon Rate  
- Call Volume by Queue  
- Agent Performance

---

# Tech Stack

- Python  
- Flask  
- SQL / CSV datasets  
- Power BI / Streamlit for dashboard  
- REST APIs

---

# How It Works

1. User selects IVR options (1 for Sales, 2 for Support).  
2. System routes calls to appropriate queue.  
3. Call data is stored in database.  
4. Dashboard calculates KPIs from call logs.  
5. Management insights are generated.
